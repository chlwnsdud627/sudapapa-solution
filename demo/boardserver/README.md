    - 비밀번호 암호화
    - 로그인, 로그아웃

    - 카테고리 관리

        - 유저 정보, 게시글 제목, 게시글 내용 등
    - 게시글 검색 기능
        - 작성 유정 아이디
        - 게시글 제목, 게시글 내용 등을 통해 검색
        - 태크 작성 및 조회 기능
    - 댓글 작성 기능
    - 대댓글
- 어드민
    - 공지글 추가 기능
